# mylink

## 📋 Overview
A modern link management platform built with Next.js and TypeScript. It allows users to create and share personalized link collections with beautiful, customizable profiles.

## ✨ Features

- **Modern UI**: Clean and intuitive interface built with TailwindCSS.
- **Link Management**: Create, edit, and organize links with custom titles and descriptions.
- **Profile Customization**: Upload profile images and set custom backgrounds.
- **Full-Stack**: Built on Next.js with TypeScript for type safety and performance.
- **Simple Deployment**: Ready for deployment on Vercel or similar platforms.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd mylink
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

### Build & Deploy

Build the application:
```bash
npm run build
```

Start the production server:
```bash
npm run start
```
